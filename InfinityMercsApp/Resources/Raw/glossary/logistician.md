# Logistician - Deployment Skill

[Optional](https://infinitythewiki.com/Labels#Optional)

## Requirements

* The user must have the **Baggage** Special Skill.
* The user must not be in a **Null**, **Isolated**, or **Immobilized** state.
* The Logistician Special Skill must not have been used previously by this army during the game.

## Effects

* At any point during the game, the user may refill an unloaded Trooper anywhere on the map without spending an Order.
* Once this Special Skill has been used by an army, that army cannot use it again for the remainder of the game.
