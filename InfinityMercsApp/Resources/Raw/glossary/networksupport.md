# Network Support - Deployment Skill

[Optional](https://infinitythewiki.com/Labels#Optional)

## Requirements

* The user must have an **EVO Hacking Device**.

## Effects

* During the Deployment Phase, the user may activate any single **EVO Hacking Program** anywhere on the field without spending an Order.
* Normal EVO hacking rules apply (for example, activating another program with this user would cancel the existing one).
