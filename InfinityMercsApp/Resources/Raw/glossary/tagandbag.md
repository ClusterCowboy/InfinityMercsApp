# Tag and Bag - Short Skill

Attack, [Optional](https://infinitythewiki.com/Labels#Optional)

## Requirements

* The user must be in silhouette contact with the target Trooper.
* The target must not have an Artichoke Token.

## Effects

* Mark a unit in the **Unconscious**, **Immobilized**, or **Shasvastii** state with an Artichoke Token. Once they are marked, this skill cannot be used against them again for the remainder of the game.
* Inflict the **Imm-A** or **Imm-B** state on that unit.
