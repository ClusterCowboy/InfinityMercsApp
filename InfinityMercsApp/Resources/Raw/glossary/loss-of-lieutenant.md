Loss of Lieutenant is not a rule used in Mercs. 

Instead, follow the rules below: 

* If the current Lieutenant enters Null State, the role is immediately transferred to the unit with the highest WIP deployed in the contract, at the end of the same order. In case of a tie, the player possessing the Lieutenant chooses how to break it. There is no other restriction; e.g. REMs may be LT’s. 
* If a unit with a WIP higher than the current lieutenant leaves the Null State, the Lieutenant role is automatically applied to him/her, at the end of the order in which the unit is healed.
* If the Captain leaves the Null State, the Lieutenant role is automatically applied to him/her, at the end of the order in which the unit is healed/restored. This is applied regardless of the WIP value of the other units deployed in the contract (ie: if the Captain is alive, he/she is always the leader, even if other units have higher WIP).
