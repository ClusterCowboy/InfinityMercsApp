# Regeneration - Automatic Skill

[States Phase](https://infinitythewiki.com/Game_Sequence#States_Phase), [Optional](https://infinitythewiki.com/Labels#Optional)

## Activation

* Automatic during the active players States Phase, if the player decides to use this Special Skill

## Effects

* During the States Phase, the user makes a Normal PH Roll:
	* If the Roll is a success, the Trooper recovers 1 lost Wounds/STR point.
* This Special Skill remains active even when its owner is in a Null state.
* This skill replaces the roll done for Healing and Recovery
