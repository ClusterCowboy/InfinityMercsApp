# Scan - Short Skill

Attack, [Optional](https://infinitythewiki.com/Labels#Optional)

## Requirements

* The user must be able to draw **LoF** to the target Trooper.
* The target must not have an Artichoke Token.

## Effects

* Mark a unit in the **Unconscious**, **Immobilized**, or **Shasvastii** state. Once they are marked, this skill cannot be used against them again for the remainder of the game.
* Inflict the **Targeted** state on that unit.
