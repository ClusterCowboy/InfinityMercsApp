# Counterintelligence - Automatic Skill

[Optional](https://infinitythewiki.com/Labels#Optional)

## Requirements

* Counterintelligence has not been used by this army this game

## Effects

* Once per game, this Special Skill counteracts **all** the effects of a single use of a Command Token.
* When a Command Token use is stopped, the Command Token is lost and another Token **may not** be used for the same purpose. (E.g. stripping orders, holding additional reserves, or a coordinated order on the order the skill was used)
